## How to Generate PDF

1. Open `executive-report-appsec-rmit.html` in Chrome
2. Click **"Print / Save PDF"** button (top-right)
3. In print dialog:
   - Destination: **Save as PDF**
   - Paper size: **A4**
   - Margins: **Default**
   - **Enable "Background graphics"** (required for colors)
4. Save

## Key Highlights

### Executive Summary Stats
- **35%** of RMIT controls have AppSec relevance
- **8 of 10** largest breaches from app-layer failures
- **$5B+** combined settlement costs (2017-2024)
- **$4.88M** average breach cost (2024)

### Investment Summary
- **Year 1 Total**: $600K-1.2M implementation + $550K-1.1M operating
- **Priority Tools**: SAST, DAST, SCA (P1), DevSecOps, Pentest (P2)

### Maturity Framework
- Primary: **BSIMM15** (Black Duck)
- Supply Chain: **Supply Chain MAP** (Black Duck)
- Open Standards: SLSA, OWASP SAMM

## Customization

To customize for your organization:
1. Replace `logo.png` with your company logo
2. Update company name in cover page
3. Update actual gap assessment ratings (Page 6)
4. Adjust investment estimates based on vendor quotes
5. Update document control table

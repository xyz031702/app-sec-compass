# Chapter 15: Security Awareness and Education - AppSec Mapping

## Control 15.1
**Type:** S (Standard)
**Requirement:** Regular technology and cybersecurity awareness education for all staff.

### AppSec Mapping
**Partial.** Only secure coding training for developers is AppSec.

| Category | Tool/Practice | How It Addresses the Control |
|----------|--------------|------------------------------|
| Secure Coding Training | Secure Code Warrior, Codebashing | Developer security training |
| Security Champions | Champion program | Embedded security advocates |

**ROI Notes:** General phishing/security awareness is NOT AppSec - use HR training platforms.

---

## Control 15.2
**Type:** S (Standard)
**Requirement:** Continuous training for staff in technology operations, cybersecurity.

### AppSec Mapping

| Category | Tool/Practice | How It Addresses the Control |
|----------|--------------|------------------------------|
| Secure Coding Training | Secure Code Warrior, OWASP WebGoat | Hands-on secure coding |
| Secure Coding Training | SANS DEV courses | Professional AppSec training |

**Notes:** Certifications: CSSLP, GWEB, CASE.

---
